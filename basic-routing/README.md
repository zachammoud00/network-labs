# Basic Routing Labs

## Overview

This section contains hands-on networking labs focused on basic routing concepts commonly used in  networks. The labs are designed to explore routing concepts including static routing, default routing, ip addressing, subnetting and troubleshooting basic connectivity issues in a network. 

### Topics Covered

- static routing

- default routing

- ping command

- subnetting

- basic LAN and WAN routing


#### Notes
These labs align primarily with CCNA-level routing fundamentals but are intended as a long-term reference as skill progress.
