# Basic Security Labs

## Overview
This section contains hands-on networking labs focused on fundamental device security concepts commonly used in enterprise networks. 
The labs are designed to demonstrate practical configuration, verification, and understanding rather than just command memorisation. 

## Topics Covered
- Enable password vs enable secret

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI

## Notes
These labs align primarily with CCNA-level security fundamentals but are intended as a long-term reference as skill progress.
