# Basic VLAN Labs

## Overview 
This section contains hands-on networking labs focused on fundamental VLAN concepts. The labs are designed to demonstrate practical configuration, verification and understanding rather than just command memorisation. 

## Topics Covered

Multi-VLAN swtich configuration
Inter-VLAN routing
VLAN Trunking 
Exploring native VLANs
VLAN port negotiation 
Voice VLAN configuration
